## 从源码构建

### 安装依赖

```bash
pnpm install
```

### 开发调试

在思源笔记调试插件

```bash
pnpm dev
```

调试博客

```bash
pnpm blogDev
```

### 生产构建

构建思源笔记版本

```bash
pnpm build
```

构建 vercel 平台版本

```bash
pnpm vercelBuild
```

构建 docker 平台版本

```bash
docker compose up --build
```
