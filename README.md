## Build from source

### Setup

```bash
pnpm install
```

### Development

for siyuan plugin

```bash
pnpm dev
```

for blog

```bash
pnpm blogDev
```

### Production

for siyuan plugin

```bash
pnpm build
```

for vercel

```bash
pnpm vercelBuild
```

for docker

```bash
docker compose up --build
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
